# Next.js Rules

This is not the Next.js many agents were trained on.

- The project uses Next.js `16.2.2` with React `19.2.4`.
- APIs, conventions, and file structure may differ from older versions.
- Before changing Next.js behavior, read the relevant guide under `node_modules/next/dist/docs/`.
- Follow current deprecations and prefer the documented local behavior over remembered patterns.
- Assume the App Router is the only routing model in use unless the code clearly shows otherwise.
