# Read First

This directory contains the project-specific rules and working context for AI agents.

Mandatory behavior:
- Read every `.md` file in this directory in ascending alphanumeric order before starting work.
- Treat these files as complementary instructions, not alternatives.
- Revisit the relevant file before touching the framework layer, routing, auth, data access, or deployment behavior.
- Add new rule files with numeric prefixes so the reading order stays explicit.
- Keep rules concise, actionable, and specific to this repository.
